Uso de la API del Covid19 con Pandas¶

Vamos a hacer un ejercicio a partir de la siguiente URL: https://api.covid19api.com/countries

El trmino API es una abreviatura de Application Programming Interfaces, que en espaol significa interfaz de programacin de aplicaciones. Se trata de un conjunto de definiciones y protocolos que se emplea para desarrollar e integrar el software de las aplicaciones, permitiendo la comunicacin entre dos aplicaciones de software a travs de un conjunto de reglas.
